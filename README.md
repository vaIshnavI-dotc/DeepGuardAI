# DeepGuardAI 🎙️🛡️

DeepGuardAI is an AI-powered deepfake voice phishing detector built using Mel-spectrograms and ResNet18 transfer learning. The system includes Grad-CAM explainability to visualize model attention during predictions.

## 🚀 Features
- Detects Real vs AI-Generated voices
- Threat meter with confidence scoring
- Waveform and Spectrogram visualization
- Grad-CAM attention heatmap
- Streamlit interactive UI

## 🧠 Tech Stack
- PyTorch
- ResNet18
- Librosa
- OpenCV
- Streamlit
- ASVspoof 2019 Dataset
